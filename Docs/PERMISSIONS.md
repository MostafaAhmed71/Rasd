# مصفوفة الصلاحيات — بوابة أعضاء هيئة التدريس

## الأدوار

| الدور | المسمى | الوصف |
|---|---|---|
| عضو هيئة تدريس | User | يدخل على مقرراته فقط، يرصد الدرجات، يقدّم طلب اعتذار، يشاهد جدوله |
| منسق برنامج | Program Admin | إشراف كامل على أعضاء برنامجه فقط + صلاحيات العضو |
| المدير التنفيذي | Super Admin | نظرة شاملة على كل البرامج والمنسقين والتقارير وإدارة الحسابات |

## المصفوفة

| الوظيفة | عضو هيئة تدريس | منسق برنامج | المدير التنفيذي |
|---|---|---|---|
| رصد الدرجات | إدخال فقط | عرض وتعديل | عرض تقارير مجمّعة |
| طلبات الاعتذار والغياب | إرسال طلب | قبول / رفض | عرض إحصائيات |
| الجداول (دراسي / مراقبة) | عرض فقط | رفع وتعديل | رفع وتعديل |
| نطاق الرؤية | مقرراته | أعضاء برنامجه فقط | كل البرامج في الكلية |
| إدارة المستخدمين | — | أعضاء برنامجه | كل المنسقين والأعضاء |
| التقارير | — | تقارير البرنامج | تقرير شامل + تصدير |

## مسارات التطبيق

- عضو التدريس: `/instructor/*`
- منسق البرنامج: `/coordinator/*`
- المدير التنفيذي: `/executive/*`
